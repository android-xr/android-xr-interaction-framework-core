# Changelog

All notable changes to the Android XR Interaction Framework will be added
here. It includes changes that affect public APIs, samples and runtime
behaviors.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this package adheres to
[Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [0.9.0] - 2024-12-12

This is the first release of the **Android XR Interaction Framework** core library. This version supports the following features:

-   Hand Raycasting
-   Gaze & Pinch (either with eyes or head)
-   XR Controllers
-   Spatial Mouse
-   Direct Touch (Poke)
-   Automatic Transitions
